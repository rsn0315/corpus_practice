그래서 탄수화물만 있는 식단을 먹더라도 되게 가려 먹어야 되고 결국에 / 한국에서 비건을 하기 위해서는 / 스스로 요리를 해서 싸 갖고 다니던지 / 하면서 먹어야 되는데 이런 게 한국에서 비건이 되기 쫌 힘든 장벽이 아닐까 이런 생각을 했었어. / 

> P2
나도 가려 먹는다는 얘기를 하니까 생각이 났는데 / 예전에 중학생 때 한 삼 개월 동안 밀가루를 끊었던 적이 있었어. / 
그런데 밀가루를 끊는 거는 채소를 끊는 것 / 가 육류나 다른 걸 끊는 것보다는 / 이제 쉬운 일이라고 생각을 보통 하는데 / 그것도 꽤 어렵더라고 외식 / 메뉴에서 밀가루를 끊고 그리고 학교를 다닐 때였으니까 / 이제 급식에서 밀가루가 / 빠지면은 먹을 게 별로 없더라고 / 그래서 / 어 비건 식을 하는 사람들이나 건강상 그렇게 생각해서 먹는 사람들도 / 응 외식을 하기 정말 어려울 것 같다고 생각해. /
