	 그런데 어느 순간부터 이런 게 안 나오니까는 너무 / 아쉽다고 생각해 히든싱어라던지 / 그런 쪽에서 나오긴 하는데 / 유독 밴드 음악에서는 잘 안 나온단 말이야. /
	 트로트 쪽도 보면은 잘 나오고 있잖아. /
	 밴드 음악이라던지 힙합이라던지 / 우리나라 쪽에서 / 이런 밴드 음악도 많이 성행을 했으면 좋겠는데 / 밴드 음악이 잘 안 나오 / 안 나오는 것도 있고 밴드 음악을 잘 안 찾는 것도 있고 / 밴드 음악을 만들기 힘들단 점도 / 우리나라에서 쪼끔 흥행하기 힘든 요소라고 생각해. /
	 당장 일본 같은 경우는 / 그~ 고등학교에서 / 악기를 하나 정도는 다루고 있다고 있다고들 하잖아. /
	
P2 	:아 물론 이제 일본에서는 원체 / 동아리 활동이 우리나라한테도 너무 유명하니까 이제 다양한 사람들이 / 하고 있겠지만 그럼에도 이제 / 우리도 시간이 지나면서 점점 / 다들 악기 하나씩 취미로 / 하는 사람들이 점점 늘어나니까 난 / 시간이 조금만 더 있으면은 / 밴드 쪽이라든가 그런 마이너 쪽은 괜찮아질 거라고 생각을 하고 있고 / 그래서 언젠가 그런 날이 오면은 이제 뭐 / 서로 원하는 곡 취향 가지고 서로 얘기하느라 아마 / 날밤 새고 그러고 있을 것 같긴 한데 / 물론 이제 밴드도 -좋- 좋지만 우리가 아직 / 다른 쪽은 분야들은 아직 얘기를 안 했잖아. /
	 그래서 뭐 다른 / 쪽 좋아하는 곡이 있으면 하나씩 꺼내주면 좋겠 / 좋을 거 같고 나 최근에는 난 그 쇼미더머니에 나왔던 / 회전 / 목마라는 노래 되게 좋아하잖아. /
