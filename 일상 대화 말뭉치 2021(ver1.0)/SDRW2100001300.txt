왜냐면 / 당연히 부모가 그런 류의 사이 / 트를 못 들어가도록 옆에서 제어해 주고 뭐 / 또 케어해 주고 / 이렇게 쭉 적당히만 하도록 이렇게 / 서포트 많이 해 준다면 / 나름대로는 이케 좋은 영양만 / 뭐 끼칠 수 있고 딱 적당량만 하는 거는 / 가능할 것 같다는 생각도 어느 정도 들긴 하는 거 같네. / 
근데 나는 인제 워낙 / 옆에서 우리 조카를 지켜보는 입장에서 / 이게 쪼끔 적당히 하는게 좋겠다 뭐 이런 생각을 많이 했었기 때문에 / 근데 아무래도 어른들도 보면 / 자기 사생활 바쁘다 보면은 / 아이한테 쓸 수 있는 시간이 쫌 적은 거 알잖아. / 
그니까 미안하니까 그만큼 뭐 자유롭게 해주어서 해줘서 / 휴대폰 / 이라도 좀 자유롭게 해서 뭐 / 적당히 뭐 재미있으라고 그런 시간 가질 수 있도록 해 주잖아. / 
그래서 / 나는 그렇게 생각을 처음에는 좀 했었거든. / 
근데 니 말을 쫌 듣다 보니까 / 부모의 역할이 쫌 어느 정도 크다는 게 쪼끔 / 많이 느껴지는 거 같고 / 뭐 나는 그런 니가 / 말해 준 그런 제어하는 뭐 그런 프로그램이 / 있고 이런 거는 처음 알아서 / 그런 거 잘 이용하면 / 이케 꼭 굳이 못하도록 / 이렇게 하는 것도 뭐 / 방법은 아닌 거 같다라는 생각이 드는 거 같애. /
