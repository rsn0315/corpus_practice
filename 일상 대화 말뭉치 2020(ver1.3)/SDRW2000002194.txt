우리 참 많이 먹으러도 다니고 / 많이 놀러도 다녔어. / 
근데 우리 항상 / 먹으르 는 거 때문에 많이도 싸우고 / 막 / 떡볶이를 먹을 건지 / 고기 먹으러 갈 건지 / 별 지금 생각해보면 별거 아닌 일인데 / 정말 많이 다퉜던 거 같애. / 
그런데 / 이제는 성인이 돼서 보니까 / 그때 다퉜던 기억도 / 지금 봐서는 되게 추억이고 / 귀엽고 / 새록새록해. / 
그때 / 우리 가르쳐주셨던 / 음악 선생님도 생각난다. / 
음악 선생님 / 노래할 때 바이브레이션 하지 말라 그러셨는데 / 계속 혼자 바이브레이션 하고 / 우리 아무것도 안 먹고 있는데 뭐 먹고 있다고 혼나고 / 그냥 우리끼리 너무 친해보이니까 / 많이 혼냈던 거 같아. /
