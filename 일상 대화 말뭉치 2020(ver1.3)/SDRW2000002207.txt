그래서 러시아에서 살다 왔는데 왜 못해 라는 / 말들을 정말 많이 들을 정도로 성적이 떨어졌고 / 러시아어를 엄청 좋아했음에도 불구하고 러시아어에 대한 반감이 생기게 된 / 거 같아. 이 학교에 진학하면서 / 물론 모든 / 친구들이 공부를 다 열심히 해서 / 나도 / 아예 안 하지는 않게 되었지만 / 정말 다 열심히 하고 잘하는 친구들이라 / 내가 / 늦게 한국에 와서 / 부족했던 거를 중학교 때는 하면 따라갈 수 있었던 거 같은데 여기에 와서는 / 거기 한계치를 느낀 거 같고 / 그거 덕분에 성 / 덕분에는 아니고 성적이 되게 많이 / 낮게 나왔던 거 같아. / 
그리고 너가 말했듯이 처음부터 기숙사 생활을 하게 되었는데 / 처음에는 / 그래도 뭔가 / 해방감 / 자유 이런 느낌이 들긴 들었지만 / 며칠 / 동안 ((이제)) 생활하면서 / 나중에는 룸메이트와 그래도 편해졌지만 / 처음에는 / 화장실을 가는 것도 눈치 보이고 해가지고 / 배도 아프고 / 스트레스도 받고 해서 / 조퇴하는 경우가 많았던 거 같아. / 
처음 반에 기숙사생들이 / 그리고 또 한 가지를 꼽자면은 / 일단은 / 친구들이 성격이 많이 맞지는 않는 거 같아. / 
특히나 / 나는 러시아어 / 과고 / 단일어과 / 이기 때문에 / 반이 바뀌는 경운 없거든 / 같은 친구들과 / 삼 년간 같은 반을 해야 되는데 / 한 번 틀어지면 힘들잖아. / 
근데 내가 / 반 친구들과 한번 / 틀어졌던 경험이 있어가지구 / 그게 이제 삼 년 동안 힘들어가지고 계속 / 지지고 볶고 하는 게 / 나한테는 엄청 큰 스트레스였던 거 같아. /
