걔가 서구적으로 생겨서 / 저 아이가 저거를 / 소화할 수 있을까 했는데 소화를 시키더라고 / 김준호가 그거를 커버를 해 주더라고 / 그래서 / 고런 류의 사극이 또 한 번 나왔으면 좋겠는데 안 나와서 아쉬워. / 

> P1
언니 근데 그 제작에 뒷 배경이 있거든? / 
근데 원래는 주인공이 걔네 둘이 아니었대. / 
근데 다른 주인공이었는데 / 걔네가 인제 / 어 차일피일 이유로 해서 하차하면서 / 어 온라인 투표를 했는데 / 이세영이 거기에 일이 위를 했더래. / 
그 배역이 너무 잘 어울린다고 근데 다행히 / 연이 닿아서 걔가 / 해 갖고 대박 히트를 쳤는데 / 나는 내가 아는 정조는 그런 쪽으로는 / 는 상상을 못했었는데 / 그 로맨스적인 거를 / 거기서 보구 내가 유튜브에서도 찾아보고 배웠던 거 같애. / 

> P2
정조를 / 다뤘잖아 언니. / 
나도 정조를 되게 유심하게 보는 편인데 / 정조 같은 경우는 정말로 / 그 여자를 싫어했어. 싫어했는데 / 유일하게 ((승은 상궁이)) 그 / 여자거든 그래서 그거를 / 어 스토리 삼아서 했는데 나는 / 아까도 말했지만 이준호가 너무 섹시하게 / 정조 역할을 너무 잘해서 나는 / 그걸로 봤던 거 같애. / 

> P3
근데 / 이준호가 가수 출신이잖아. / 
근데 가수 출신치고는 연기는 잘하는데 / 너무 이미지를 팔아 먹는 느낌? / 
거기에서 그 붉은 끝동에 나온 이미지로 막 / 광고를 너무 많이 나오니까 / 뭐라 그러지 그 / 끝동의 이미지가 / 다 사라지지도 않았는데 / 현실 세계로 돌아온 느낌 / 약간 / 어느 정도는 쉬었다가 나왔으면 / 더 오래 갔을 거 같은데 아쉽더라고 /

걔도 먹고 살아야 될 거 아냐. / 
연예인이 뭔데 / 그런 것 때문에 수익을 올리는 건데 나는 괜찮다고 봐. / 
이준호 그리고 / 아이가 서글서글하고 뭐 어디 뭐 / 어 나쁜 뭐 스캔들에 뭐 휩싸이지도 않았고 / 나는 걔 괜찮거든 그래서 팬 카페도 / 뭐 사람들 많이 가서 하고 / 음 요즘에 근데 좀 어디 나오니 걔? / 

> P1
어 / 차기작은 확정이 됐다 그러더라고 근데 / 걔가 그전에 드라마 찍었을 때 자백인가 하고 / 또 영화도 찍은 걸 내가 / 끝동 보고서 너무 얘가 연기를 잘해서 찾아서 봤는데 / 그 배역 배역마다 너무 잘해서 오히려 더 / 기존 이미지보다 더 좋아졌던 거 같애. / 

> P2
